# Data Visualization Dashboard

This project presents a sales performance dashboard created using Power BI.

## Key Insights
- West region leads overall sales
- Profit trend shows recent decline
- Technology category contributes highest revenue
- Revenue is concentrated among top customers

## Tools Used
- Power BI
- Dataset: Superstore Sales

This dashboard demonstrates data visualization, storytelling, and business insight analysis.
